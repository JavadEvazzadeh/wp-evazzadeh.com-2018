پلاگین اختصاصی ابر آروان - حذف اتوماتیک کش

== Description ==

افزونه حذف خودکار کش ابرآروان به شما کمک می‌کند Cache سایت وردپرسی خود را به شکل اتوماتیک حذف کنید و محتوای جدید سایت، شامل پست‌ یا دیدگاه را در لحظه به کاربران‌تان نمایش دهید.
خدمات CDN آروان می‌تواند بهترین گزینه برای استفاده از CDN در سایت‌های وردپرسی ایرانی باشد. مهم نیست شما از خدمات هاستینگ چه شرکتی استفاده می‌کنید، CDN آروان با خدمات تمامی هاستینگ‌های ایران و خارجی سازگاری دارد و با داشتن چندین پاپ سایت‌های اختصاصی در داخل ایران، به مراتب عملکرد بهتری نسبت به CDN های خارجی دارد برای کاربران و بازدیدکنندگان ایرانی دارد.

فعال کردن شبکه توزیع محتوا (CDN) آروان این مزیت‌ها را برای سایت شما به همراه دارد:
"	افزایش سرعت لود صفحات
"	کاهش زمان پاسخگویی
"	کاهش نرخ پرش
"	کاهش حجم محتوای صفحات
"	بهبود سئو سایت
"	افزایش امنیت در برابر حملات DDoS
"	صرفه‌جویی در مصرف ترافیک
"	دسترسی به سایت حتی در زمان دان شدن سرور اصلی
"	دریافت گزارشات تحلیلی و آماری



== Installation ==

1. Upload arvancloud-cache-cleaner folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a ArvanCloud account at https://www.arvancloud.com/register
1. Enter your ArvanCloud Account & API token & API Domain & Domain on the plugin settings page
